# Kanban Board

A Kanban board is a visual tool that helps manage tasks and workflows. This project is a digital implementation of a Kanban board to track tasks, manage workflows, and improve productivity.

## Features

- **Create Boards**: Easily create and manage multiple Kanban boards.
- **Add Lists**: Organize tasks into lists (e.g., To Do, In Progress, Done).
- **Drag and Drop**: Move tasks between lists using a drag-and-drop interface.
- **Task Management**: Add, edit, and delete tasks with detailed descriptions and deadlines.
- **Customizable**: Personalize board colors, list names, and task details.

## Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/harsh-791/Kanban-Board.git
   ```

2. **Navigate to the Project Directory**
   ```bash
   cd Kanban-Board
   ```

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Start the Application**
   ```bash
   npm start
   ```

   The application will be available at `http://localhost:3000` in your browser.

## Usage

1. **Create a Board**: Click on "Create New Board" and enter a name for your board.
2. **Add Lists**: Click on "Add List" to create new lists and categorize your tasks.
3. **Add Tasks**: Click on "Add Task" within a list to create tasks. Enter task details and set deadlines if necessary.
4. **Drag and Drop**: Drag tasks between lists to update their status and track progress.
5. **Edit and Delete**: Click on tasks to edit details or delete them as needed.
